# Better Backgrounds

Better Backgrounds is a pre-implementation design for a cross-platform desktop
application that reconstructs an empty room from video, lets the user choose a
plausible background viewpoint, and uses the reconstructed scene's spatial and
view-dependent information to harmonise a live webcam foreground with that
view.

The application has not been implemented yet. Product and architecture planning
documents are maintained locally under the ignored `docs/` directory.
